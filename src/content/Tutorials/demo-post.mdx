This page demonstrates some of the basic features and functionality of this help site.

Since this is MDX, we can use a `<Tip>` component for helpful suggestions.

<Tip>
  Here is some extra information that you might find helpful. However, feel free
  to ignore it.
</Tip>

There is also a `<Warning>` component for more serious information.

<Warning>Don't ignore this! It is important information!</Warning>

We also have a custom codeblock component so that we can have nice syntax highlighting in code samples. Click the clipboard to copy the code.

You can change the syntax highlighting theme based on the overall site theme as well. Try clicking the Sun/Moon icon at the very top of the page to see.

```js
let sandwich = "grilled rachel";

  let three = 1 + 2;
  return three;
}
```
